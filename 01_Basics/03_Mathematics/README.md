## 📌 Concepts Covered

### 🔹 1. GCD (Greatest Common Divisor)

* Finds the largest number that divides two numbers
* Implemented using the **Euclidean Algorithm**

---

### 🔹 2. LCM (Least Common Multiple)

* Smallest number divisible by both numbers
* Relation:

```id="math1"
LCM(a, b) = (a × b) / GCD(a, b)
```

---

### 🔹 3. Prime Numbers

* A number divisible only by **1 and itself**
* Optimized check runs in **O(√n)**

---

### 🔹 4. Sieve of Eratosthenes

* Efficient way to find all primes up to `n`
* Time Complexity: **O(n log log n)**

---
